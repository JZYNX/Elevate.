# Running Locally
- Please clone the 'Local-demo' branch

## Running server: 
- cd server
- npm install
- npm run devStart

## Running frontend:
- cd crm
- npm install 
- npm run start

## Running development build (frontend & backend):
- Remain in root directory
- npm install
- npm run start-dev

# Testing
- Remain in root directory
- npm install
- npm test
